# プロ野球の来年の成績を「データだけ」で予測するシステムを作った

「来年、このチームは何位だろう？」
「あの選手はどのくらい活躍するだろう？」

こんな疑問にデータで答えてみようと、NPB（日本プロ野球）の2015年〜2025年・全選手の成績をもとに、2026年の予測システムを作りました。

Streamlitで公開しているので、誰でもブラウザで触れます。
https://npb-prediction.streamlit.app/

> **この記事に出てくる用語**
> - **OPS** — 出塁率＋長打率。打者の総合力を測る指標。.700台が平均、.900超えは一流
> - **ERA（防御率）** — 投手が9イニングあたりに取られる自責点。小さいほど優秀
> - **wOBA** — 打者の1打席あたりの得点貢献を測る指標。打率よりも正確に実力がわかると考えられている
> - **wRC+** — wOBAをリーグ平均=100に換算したもの。150なら「平均の1.5倍の得点力」

## 予測の仕組み（3段階）

### 第1段階: Marcel法 — 「過去3年の加重平均」

Tom Tangoが1980年代に考案した、シンプルだけど強力な手法です。

1. 過去3年の成績を「5:4:3」の重みで平均する
2. 試合に出ていない選手はリーグ平均に近づける
3. 年齢で微調整する（27歳前後がピークと考えられている）

たったこれだけで、2024年・2025年の2年分の検証では最新のAI（LightGBM/XGBoost）より一貫して正確でした。

### 第2段階: Stan/Ridge補正 — 「スキル指標で微調整」

Marcel法は「全員同じように平均に引き戻す」という弱点があります。

例えば、三振が少なくて四球が多い打者（打席での判断が良い打者）は、実力が本物である可能性が高いと考えられます。こうした「実力の裏付け」を数値化して、Marcel法の予測を微調整しています。

使っている指標:
- **K%（三振率）** — 低いほど打席での安定感がある傾向
- **BB%（四球率）** — 高いほど選球眼が良い傾向
- **BABIP** — インプレーの打率。極端な値は「運」の要素が大きい可能性
- **年齢** — ピーク前 vs ピーク後で予測を調整

この補正は8年間分のデータ（2018-2025年）で検証し、統計的に有意な改善を確認しています。

### 第3段階: ピタゴラス勝率 — 「チームの強さ = 得点力 - 失点」

各選手の予測を合算して、チームの得点と失点を見積もります。「得点が多くて失点が少ないチームが勝つ」という法則を数式にしたピタゴラス勝率で、チーム順位を予測します。

## 予測の限界

このシステムには限界もあります。

- **若手の急成長を捉えにくい**: Marcel法は過去3年の平均なので、ブレイク直前の選手の予測は低く出がちです
- **新外国人選手は予測できない**: NPBでのデータがないので、リーグ平均値を当てはめています
- **ケガの予測はしていない**: 「フルシーズン出場できれば」という前提の予測です

こうした限界を理解した上で見ると、データ分析の面白さが伝わるのではないかと思います。

## 技術的な話（興味がある方向け）

- **データ**: プロ野球データFreak / NPB公式サイトから2015-2025年の全選手成績を収集
- **Marcel法**: 過去3年の成績を5:4:3の重みで加重平均 → リーグ平均へのregression → 年齢調整
- **Stan/Ridge補正**: Ridge回帰（α=1.12 打者 / 4.84 投手）でMarcel予測にスキル指標ベースの補正を加算
- **ピタゴラス勝率**: RS^1.72 / (RS^1.72 + RA^1.72)（NPB最適化済み）
- **検証**: 8年間のLOO-CV（Leave-One-Out交差検証）で統計的有意性を確認
- **フロントエンド**: Streamlit（日本語/英語切り替え対応）

コードはすべてGitHubで公開しています。

→ https://github.com/yasumorishima/npb-prediction

## まとめ

- NPB全選手の過去10年の成績から、来年の予測を自動計算するシステムを作った
- Marcel法（1980年代のシンプルな手法）が基盤で、スキル指標による補正を加えている
- チームの順位予測まで一気通貫でできる
- Streamlitで公開しており、ブラウザで誰でも触れる

「予測」は当たったり外れたりするものですが、「なぜそう予測されるのか」を理解するプロセスが、データ分析の醍醐味だと感じています。

データの提供元：[プロ野球データFreak](https://baseball-data.com) / [NPB公式サイト](https://npb.jp)
